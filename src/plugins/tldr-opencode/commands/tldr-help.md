Show the TLDR quick-reference card.

| Command | What |
|---|---|
| `/tldr` | Activate at default level (full) |
| `/tldr lite` | Light compression — ~30% tokens dropped |
| `/tldr ultra` | Maximum compression |
| `/tldr wenyan[-lite\|-ultra]` | Classical Chinese compression |
| `/tldr off` | Deactivate |
| `/tldr-commit` | Terse commit message |
| `/tldr-review` | One-line review findings |
| `/tldr-compress <file>` | Compress a Markdown file |
| `/tldr-stats` | Lifetime token-savings |

Natural language also works: "turn on TLDR", "stop tldr", "normal mode".

Code, commits, security warnings: TLDR drops out automatically.
